# JS Station
The sensor station for the JavaScript platform. You can run the sensor station in cross-platform(Mac, Linux, and Windows).

## How to Run
1. Clone the repo.
2. Go to the `js-station` directory.
3. Do `npm install`.
4. Do `node app.js`.
5. Done.
